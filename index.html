Projects

Projects I have worked on in research, classes, and for fun

Engineering line of fluorescent reporter mESCs

Genetically engineering a line of mESCs to report X-chromosome activity upon induced X-chromosome inactivation. Building gene constructs using PCR and Gibson Assembly

Image processing pipeline for RNAseq

Building a pipeline to take fluorescent images, segment cells based on their nuclear signal, determine if XIST expression found, and how many active X-chromosomes are present based on the mRNA signal.

Improvement of enzymatic Si-C bond cleavage

Using site saturated mutagenesis of residues val184 and leu188 to improve the oxidative activity of P450 BM3 in breaking down volatile methylsiloxanes.

Pulse Oximeter

Built a working pulse oximeter using an Arduino and breadboard, as well as code to operate the pulse ox, signal processing, and an interface through Python.

Modeling the Interaction Between Viruses and Autoimmune Disorders

Modeled the interactions between viruses and the human immune response using differential equations and used Berkeley Madonna to visualize the interactions. Gave a presentation on the topic

Treating Chronic Epstein-Barr Virus Infections through RNAi

Did research on using RNAi to trigger an immune response during Epstein-Barr virus reactivation, gave a presentation, and wrote a proposal.
